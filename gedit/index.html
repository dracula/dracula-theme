Dracula

A dark theme for Gedit and more

Gedit

Install manually

First, download the raw file from the repo

$ wget https://raw.githubusercontent.com/zenorocha/dracula-theme/master/gedit/dracula.xml

Then, move the downloaded file to Gedit style's folder

$ mv dracula.xml $HOME/.local/share/gedit/styles/

Activating theme

Open Gedit. Go to Preferences -> Fonts & colors and select Dracula theme from list.
