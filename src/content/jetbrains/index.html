JetBrains

This theme works with all JetBrains IDEs like IntelliJ, WebStorm, PyCharm, PhpStorm, etc.

Whenever you see a {NameOfIDE}, translate it to the IDE you're using.

Install using Git

If you are a git user, you can install the theme and keep up to date by cloning the repo:

$ git clone https://github.com/zenorocha/dracula-theme.git

Install manually

Download using the GitHub .zip download option and unzip them

Activating theme

  1. Copy the ./jetbrains/Dracula.icls file to the ~/Library/Preferences/{NameOfIDE}/colors directory on your local machine.
  2. Close and re-open {NameOfIDE}.
  3. Navigate to {NameOfIDE} -> Preferences -> Editor -> Colors & Fonts.
  4. Select “Dracula” from the dropdown menu.
  5. Click the “Ok” button to activate the color scheme.
